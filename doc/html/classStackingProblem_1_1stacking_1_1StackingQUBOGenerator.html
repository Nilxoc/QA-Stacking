Generate term for f(t,c), which indicates whether bins with label t require a stacking place at time c.

After execution, bqm will contain a variable 'f(t,c)' for every c and constraints will be modeled so 'f(t,c)' contains 1 if a stacking place is required and 0 if no stacking place is required.
